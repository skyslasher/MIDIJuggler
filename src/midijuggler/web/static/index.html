            OSC Libraries

              MIDI Libraries

                Monitor

                  System

                  Identify this box on the network and recover without power cycling.

                  Pause desk-to-controller encoder feedback briefly after encoder turns. 0 disables.

                  Devices

                  Logical devices bind one I/O adapter instance to a library for routing and connections. Each adapter can be bound to at most one device.

                  OSC

                  Desk libraries (X32/Wing) use the same local and remote OSC port and renew subscriptions every 9 seconds. Linked desks are re-discovered in the background; Wing uses its serial number, X32 uses its network name as stable identity.

                  Wing Native

                  Direct TCP connection to the Wing desk (port 2222). Parameter paths come from the Device bound to each instance. Up to 16 native clients can connect simultaneously; OSC event subscription is separate.

                  MIDI

                  ALSA MIDI ports for each instance. The control library (e.g. X-Touch Mini) is configured on the bound Device. X-Touch Mini also shows LED feedback refresh and MIDI channel options here when the bound Device uses that library. Echo guard ignores input that matches recently sent output on the same path (MIDI loopback or OSC desk echo). Set to 0 to disable. Default is 30 ms.

                  RTP-MIDI

                  HID Inputs

                  Map Linux evdev controls from gamepads and USB devices. Save and enable an instance, then use Learn input to capture buttons and axes.

                  GPIO Inputs

                  MIDI Master Clock

                  MIDI clock outputs

                  Route BPM and transport via Connections to clock.* data points (for example clock.bpm_set, clock.bpm_huge_up, clock.click_set, clock.click_toggle, clock.click_interval_set, clock.click_interval_cycle, clock.start, clock.stop). Read clock.click_enabled for the current audio click state. Use clock.beat for a short beat flash; duration is capped to 90% of the selected click interval.

                  Rotary Display

                  Host transport controls how MIDIJuggler sends clock sync to the device. Device settings are pushed over USB serial when the device sends hello (and after saving here).

                  Device config (pushed over serial)

                  Unique per encoder. Leave empty to auto-generate from MAC (rotary-a1b2c3.local). Hello registers this name for host feedback.

                  BandHelper

                  Follow song tempo from BandHelper over Ableton Link and receive musical key from OSC presets. See docs/bandhelper.md for BandHelper setup.

                  GamePi

                  Co-located kiosk UI and brightness datapoints. Changing enabled or watch settings requires a service restart.

                  Configuration import/export

                  Export the active TOML configuration or import a saved configuration for another setup.
