              System

              Pause desk-to-controller encoder feedback briefly after encoder turns. 0 disables.

              Devices

              Logical devices bind one I/O adapter instance to a library for routing and connections. Each adapter can be bound to at most one device.

              OSC

              Desk libraries (X32/Wing) use the same local and remote OSC port and renew subscriptions every 9 seconds. Linked desks are re-discovered in the background; Wing uses its serial number, X32 uses its network name as stable identity.

              Wing Native

              Direct TCP connection to the Wing desk (port 2222). Parameter paths come from the Device bound to each instance. Up to 16 native clients can connect simultaneously; OSC event subscription is separate.

              MIDI

              ALSA MIDI ports for each instance. The control library (e.g. X-Touch Mini) is configured on the bound Device. X-Touch Mini also shows LED feedback refresh and MIDI channel options here when the bound Device uses that library. Echo guard ignores input that matches recently sent output on the same path (MIDI loopback or OSC desk echo). Set to 0 to disable. Default is 30 ms.
